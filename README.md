# Developers

## Installation

You will need:

- [Meteor](https://www.meteor.com/install)
- [NodeJS](https://nodejs.org/en/download/)

Once installed run the following (for those using Windows, run in the NodeJS Command Line):

```sh
cd gild-game
npm install
meteor
```
